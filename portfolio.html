Website for Pet healthcare and needs

Web Dev project

The pets or domesticated animals had always been a vital part of human evolution. They helped human’s lifestyle to develop as a whole. Hence, our project aims to return back something to them as a humanitarian idea of doing something for a cause of these speechless beings.

So the basic idea is as follows:

Collecting Raw data from various sources like NGO’s, Owners’, Veterinary Doctors.

Compiling this Data to point the main problems.

Checking a feasibility test if a website or an application is already present to see this problem.

Making a Web Interface for the solution of the above mentioned problems about domestic animals’ healthcare.

Creating a Database for Pet owners & respective Doctors Separately.

Providing the consultancy to pet owners according to severity of healthcare issue of their pet.

Finally matchmaking the User to the respective consultant.

Summary of our data analysis

As we collected our data, we came to know that the availability of healthcare for domestic animals is very scarce in many regions. Especially due to Covid-19 the condition is much alarming. A need of a platform for this issue was required as we asked the general pet owners. Hence our website creates a mobile consultancy platform for them right at their home.
